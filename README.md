# DOMination Quiz

A simple JavaScript quiz application built with HTML, CSS, and JavaScript to practice DOM manipulation and core JS concepts.

## What This Project Does

- Shows one question at a time
- Lets the user choose an answer
- Gives immediate feedback (green = correct, red = incorrect)
- Tracks the score
- Displays final results
- Allows the quiz to be restarted

## Concepts Used

- Arrays and objects
- Loops (`forEach`)
- Functions
- DOM manipulation
- Event listeners
- Conditional logic

## File Structure

index.html
styles.css
script.js

## How to Run

1. Open the folder in VS Code
2. Right-click `index.html`
3. Select **Open with Live Server**
